# Finch React Demo

Easily try out Finch Connect and API requests, either in sandbox or with live data.

## Getting Started

Replace Keys in APP.js

```
let CLIENT_ID = "<<YOUR_CLIENT_ID>>";
let CLIENT_SECRET = "<<YOUR_CLIENT_SECRET>>";
```

Run the app
`yarn install`
`yarn start`
